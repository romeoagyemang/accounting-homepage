# Harrow & Cole Accountants — Homepage

Homepage for a fictional small London accounting firm, built per the Task 1 brief: reliability, responsiveness, and personal service over price, targeting sole traders and limited companies under £500k turnover.

## Stack
- Next.js 16 (App Router), TypeScript
- Tailwind CSS v4
- Self-hosted fonts (Inter + Source Serif 4) via `next/font/local` — no third-party font requests

## Run locally
```bash
npm install
npm run dev
```

## Deploy
Import this repo directly into Vercel — no environment variables required.
